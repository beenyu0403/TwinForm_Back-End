## 소개
- 간단하게 설문지를 제작하거나 주문서, 신청서를 원하는 형태로 커스텀하여 편집할 수 있도록 하고, 폼에 참여한 기록을 관리하는 iOS APP 서비스
  
## 프로젝트 목표
"앱을 통해 간단하게 필요한 폼을 제작하고, 참여한 폼을 한눈에 확인할 수 있는 서비스" 개발

## 집중 개발 분야
백엔드(Back-End) - Spring Boot 프레임워크, Java, H2 Database

## 제공 서비스 (API)
- 회원 가입 / 로그인
- 폼 생성
- 생성한 폼 조회 / 생성한 폼 편집
- 참여한 폼 조회 / 참여한 폼 내용 확인
- 회원 정보 조회
...
